  • UNC Charlotte webspace exists and loads correctly.

CRAP Design Principles

  • Contrast: Strong dark-mode theme with excellent text contrast.
  • Repetition: Consistent card, footer, and header styling applied across multiple pages.
  • Alignment: Clean centered layout; sections line up properly.
  • Proximity: Related content is grouped logically and spaced evenly.

Standard Page Requirements

  • Header includes course name and branding correctly.
  • Navigation bar is consistent across pages with working links.
  • No page name included in the header (correct for branded layout).
  • H1 used appropriately for the site/brand heading.
  • Main section begins with a clearly labeled H2 such as “Home” or “Introduction”.
  • Footer includes:
    • User’s page links
    • HTML & CSS validation links
    • Brand/company attribution (DylaTrong Designs)
  • CSS is linked properly and shows consistent use of fonts, spacing, and theme.
  • No black/white/papyrus/comic sans, correct font usage.

Assignment-Specific Requirements

  • Home, Introduction, Contract, Website Evaluations, and other course pages completed.
  • Images load properly on the Introduction page.
  • Links appear functional, including external resources (GitHub, LinkedIn, FCC, Codecademy).
  • Slogan/tagline shown on Home page (“Dynamic Turtle”).
