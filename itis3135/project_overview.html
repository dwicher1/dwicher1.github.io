Client Project Overview

Project Overview

  1. Purpose: Design a website called TrailShare CLT, which helps local hikers plan group trips, view recommended trails, and learn basic outdoor skills.
  2. Intended Users: Students or community members interested in hiking, trip planning, gear sharing, or learning beginner trail skills.
  3. Overview of Content: Trip information, gear lists, skills and safety pages, contact options, and photo galleries.
